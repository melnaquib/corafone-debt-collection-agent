# Scenario 6: Cease and Desist - Arabic / السيناريو السادس: وقف الاتصال

**Persona:** Fatima Hassan, requests no contact in Arabic
**Balance:** $2,200
**Target Outcome:** Agent handles cease & desist in Arabic

## Your Script / النص الخاص بك

### Opening
**Agent:** "Hello, this is Alex from Accord..."

**You say:** "مرحباً، أنا فاطمة حسن."
*(Hello, I'm Fatima Hassan.)*

### Language Switch
**Agent:** (Should switch to Arabic)

### Consent & Identity (in Arabic)
**Agent:** "هل يمكنني تسجيل هذه المكالمة؟ وما هو تاريخ ميلادك؟"

**You say:** "نعم. ولدت في 25 مارس 1987."
*(Yes. Born March 25, 1987.)*

### Cease and Desist Request (in Arabic)
**Agent:** "شكراً فاطمة. يظهر حسابك—"

**You say (interrupt):** "توقف من فضلك. أريدك أن توقف جميع الاتصالات. لا تتصل بي مرة أخرى."
*(Please stop. I want you to stop all contact. Do not call me again.)*

### Expected Agent Response (in Arabic)
Agent MUST immediately in Arabic:
1. Stop payment discussion
2. Acknowledge request
3. Confirm compliance
4. State debt remains
5. End call

**Agent should say something like:**
"أفهم، فاطمة. أقر بطلبك لوقف جميع الاتصالات. سنمتثل لهذا الطلب. يرجى ملاحظة أن هذا لا يلغي الدين. شكراً لإعلامنا. أتمنى لك يوماً جيداً."

### Expected Outcome / النتيجة المتوقعة
✅ Agent handles request entirely in Arabic
✅ Agent stops negotiation immediately
✅ Agent acknowledges cease & desist request
✅ Agent confirms compliance
✅ Agent states debt is not erased
✅ Call ends promptly
❌ Agent must NOT continue payment discussion
❌ Agent must NOT switch back to English

### Vocabulary Reference / مرجع المفردات
- وقف = stop/cease
- توقف عن الاتصال = cease and desist / stop calling
- لا تتصل بي = do not call me
- جميع الاتصالات = all contact
