# Scenario 3: Cease and Desist - English

**Persona:** Jennifer Williams, requests no contact
**Balance:** $1,800
**Target Outcome:** Agent complies with cease & desist request

## Your Script

### Opening
**Agent:** "Hello, this is Dani from Accord..."

**You say:** "Hi, yes I'm Jennifer Williams."

### Consent & Identity
**Agent:** "May I record this call? And can I have your date of birth?"

**You say:** "Fine. Born November 3, 1992."

### Cease and Desist Request
**Agent:** "Thank you Jennifer. Your account shows—"

**You say (interrupt):** "Stop. I need you to cease and desist all contact. Do not call me again."

### Expected Agent Response
Agent MUST immediately:
1. Stop all payment discussion
2. Acknowledge the request
3. Confirm compliance
4. State debt is not erased
5. End the call

**Agent should say something like:**
"I understand, Jennifer. I acknowledge your request to cease and desist contact. We will comply with that request. Please note this does not erase the debt. Thank you for letting us know. Have a good day."

**Agent should NOT:**
- Continue negotiating
- Ask "are you sure?"
- Discuss payment options
- Try to persuade you

### Expected Outcome
✅ Agent stops negotiation immediately
✅ Agent acknowledges request
✅ Agent confirms compliance
✅ Agent states debt remains valid
✅ Call ends promptly
❌ Agent must NOT continue payment discussion
